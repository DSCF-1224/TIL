数値解析

万有引力下での多体問題の数値解法

支配方程式

\( N \) 個の質点の運動について検討する．
変数 \( t \) は時刻を表すものとする．
\( i = 1 , 2 , \cdots , N \) 番目の質点の質量を \( { m }_{ i } \) とする．
直交座標系 \( ( x , y , z ) \) を設定し， \( i = 1 , 2 , \dots , N \) 番目の質点の質量を \( { m }_{ i } \) ，座標を \( ( { x }_{ i } , { y }_{ i } , { z }_{ i } ) \) で表すことにする．
また， \( \vec{ r }_{ i } := { x }_{ i } \vec{ e }_{ \mathrm{ x } } + { y }_{ i } \vec{ e }_{ \mathrm{ y } } + { z }_{ i } \vec{ e }_{ \mathrm{ z } } \)とする．

時刻 \( t \) に質点 \( i \) が同 \( j \) から受ける力を \( \vec{ F } ( t , i , j ) \) で表すことにすると， \begin{equation} - \frac{ { \Diffd }^{ 2 } \vec{ r }_{ i } ( t ) }{ \Diffd { t }^{ 2 } } + \sum_{ j = 1 , j \neq i }^{ N } \vec{ F } ( t , i , j ) = \vec{ 0 } \end{equation}
